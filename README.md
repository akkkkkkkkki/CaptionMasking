# Scripta - Vocab Hunter

一个 Chrome 浏览器插件，用于从 YouTube 视频字幕中提取生词，帮助你高效学习英语词汇。

## 功能特点

- 🎯 自动提取 YouTube 视频字幕中的所有单词
- 📊 智能词汇管理：新词 → 熟悉 → 已掌握
- 🔍 搜索和筛选功能
- 📚 预设常用词表（1000/3000/5000 词）
- 🎲 随机打乱词序，增加学习趣味性
- 💾 导入/导出词表（CSV 格式）
- 🌐 支持多个在线词典（牛津、剑桥、韦氏、有道、谷歌翻译）

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/yourusername/scripta-vocab-hunter.git
cd scripta-vocab-hunter
```

或者直接下载 ZIP 文件并解压。

### 2. 导入到 Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的「开发者模式」开关
4. 点击左上角的「加载已解压的扩展程序」按钮
5. 选择项目文件夹（包含 `manifest.json` 的文件夹）
6. 安装完成！��会在浏览器右上角看到插件图标

## 使用方法

### 第一步：提取字幕单词

1. 打开任意 YouTube 视频
2. **确保视频开启了字幕**（点击视频右下角的 CC 按钮）
3. 点击浏览器右上角的 Scripta 插件图标
4. 点击「🔍 扫描字幕」按钮
5. 等待扫描完成，会显示提取到的单词数量
6. 点击「📖 打开词表」按钮，在新标签页中查看完整词表

### 第二步：管理词汇

在词表页面，你可以：

#### 筛选词汇
- **All**：显示所有单词
- **New**：只显示新词（未标记的单词）
- **Learning**：显示正在学习的单词
- **Mastered**：显示已掌握的单词

#### 标记单词状态
鼠标悬停在单词上，会出现操作按钮：
- **👁️ 熟悉**：标记为"正在学习"（点击多次后自动变为"已掌握"）
- **✓ 掌握**：直接标记为"已掌握"（悬停时会显示删除线动画）
- **🔊 发音**：在线词典查询（可在设置中选择词典）

#### 搜索单词
在右上角搜索框输入关键词，实时过滤单词列表。

#### 随机打乱
点击筛选器旁边的 🔀 按钮，随机打乱单词顺序，让学习更有趣。

### 第三步：导入常用词表（可选）

如果你已经掌握了一些基础词汇，可以批量导入：

1. 点击右上角的「📚 Import Common Words」按钮
2. 选择词汇量级别：
   - **Top 1000**：最常用的 1000 个英语单词
   - **Top 3000**：高中水平词汇
   - **Top 5000**：大学水平词汇
3. 点击「Import」，这些单词会自动标记为"已掌握"

### 设置选项

点击右上角的 ⚙️ 按钮，可以调整：

- **Familiar threshold**：点击多少次"熟悉"后自动变为"已掌握"（默认 3 次）
- **Dictionary**：选择查词时使用的在线词典

### 导入/导出

- **💾 Export**：将当前词表导出为 CSV 文件
- **📥 Import**：从 CSV 文件导入词表
- **🗑️ Clear All**：清空所有数据（谨慎使用）

## 其他功能

### 字幕遮罩（可选）

在 YouTube 视频页面，点击插件图标，然后点击「🙈 切换遮罩」，可以隐藏字幕，测试自己的听力水平。

## 技术说明

- 基于 Chrome Extension Manifest V3
- 使用 `chrome.storage.local` 存储数据
- 通过拦截 YouTube 字幕 API 请求获取字幕数据
- 纯前端实现，无需后端服务器

## 常见问题

**Q: 为什么扫描后显示 0 个单词？**
A: 请确保视频已开启字幕（CC 按钮），并刷新页面后重试。

**Q: 数据会丢失吗？**
A: 所有数据存储在浏览器本地，不会上传到服务器。建议定期导出备份。

**Q: 支持其他视频网站吗？**
A: 目前仅支持 YouTube。
